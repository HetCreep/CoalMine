# rot-canary auto-cadence ‚Äî hook wiring per platform

Only Claude Code (plugin route) wires automatically. Every snippet below is **manual**: copy it into place, adjust the CoalMine path, and test in your setup ‚Äî the cadence is post-edit ‚Üí record touched file ¬∑ session-stop ‚Üí nudge a QUICK scan. Event names verified against vendor docs (Jun 2026); stdin payload field names may still differ per platform.

Response-schema honesty: a row gets a full wiring claim only where the target's RESPONSE schema (not just the request-side config format) is verified against vendor docs; otherwise the row says so and the shipped config falls back to the plain Claude-Code shape (no trailing argument) as a best-guess. Tier language stays **works with** for every platform below ‚Äî none has an empirical live-session pilot fire (the bar `antigravity-hooks.json` cleared 2026-07-12), so none is called "validated" or "wired".
